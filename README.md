# NetCp Multi-File
.

## Compilation

To compile the program, execute `make` command.

	$ make


## Usage

Execute, along with their corresponding options:

	$ ./NetCp

## Help

For more knowledge about program's functions, execute:

	$ ./NetCp --help

or

	$ ./NetCp -h

## Cleaning

Execute `make clean` for executables and objects cleaning:

	$ make clean
